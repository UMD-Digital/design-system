    Function toCssObject

    • toCssObject(cssVarsObj: Record<string, any>): string

      Converts CSS variables object to CSS custom properties object.

      Parameters

      • cssVarsObj: Record<string, any>

        CSS variables object

      Returns string

      CSS custom properties as a string

      Example

      import * as Styles from '@universityofmaryland/web-styles-library';

      const cssVars = {
        --color-primary: '#ff0000',
        --color-secondary: '#00ff00'
      };

      const variables = Styles.utilities.transform.variables.toCssObject(cssVars);
      // Result: { '--color-primary': '#ff0000', '--color-secondary': '#00ff00'};
